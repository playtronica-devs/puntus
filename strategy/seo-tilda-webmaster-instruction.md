## 1. Оптимизация существующих категорий (L1, L2) — СЕГОДНЯ

Tilda: страница → **Настройки страницы** (шестерёнка) → вкладка **«Главное»** (Title/Description/URL).

### L1 — `/rings` (кольца с жемчугом) 🔥 приоритет
- **Title:** `Кольца с жемчугом ручной работы — Puntús`
- **Description:** `Кольца с живым речным и барочным жемчугом. Ручная работа, каждое — единственное. Доставка по России.`
- **URL:** оставить `/rings` (НЕ менять).
- На странице: **H1** = `Кольца с жемчугом`; интро: `Живой жемчуг — неидеальный, а значит настоящий. Каждое кольцо мы делаем вручную, поэтому второго такого нет.`
- 200–400 слов уникального текста (жемчуг/камни/ручная работа, тёплый голос).
- **JSON-LD:** Настройки страницы → **«Дополнительно»** → **«Код внутри HEAD»**:
```html
<script type="application/ld+json">
{"@context":"https://schema.org","@type":"CollectionPage","name":"Кольца с жемчугом ручной работы","brand":{"@type":"Brand","name":"Puntús"}}
</script>
```

## 4. Ускорить индексацию (после публикации каждой страницы)
1. **Яндекс.Вебмастер** → Индексирование → **«Переобход страниц»** → вставить URL.
2. **GSC** → «Проверка URL» → **«Запросить индексирование»**.
3. Проверить, что URL попали в `sitemap.xml`.

## 5. YML-фид (когда дойдут руки)
Tilda Store → фид → включить **YML** → ссылку в Яндекс.Вебмастер → Товары и цены. Вывод карточек в товарной выдаче Яндекса.
